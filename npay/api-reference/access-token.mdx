# Get Access Token

Get an access token to initiate payment transactions. This endpoint authenticates your API credentials and returns a temporary access token.

<Note>
  Access tokens expire after 600 seconds (10 minutes). Implement token refresh logic in your application.
</Note>

## Request

<RequestExample>
```php
<?php
require_once('vendor/autoload.php');
$client = new \GuzzleHttp\Client();
$response = $client->request('POST', '{{base_url}}/authentication/token', [
    'json' => [
        'client_id' => 'your_client_id_here',
        'secret_id' => 'your_secret_key_here',
    ],
    'headers' => [
        'accept' => 'application/json',
        'content-type' => 'application/json',
    ],
]);
echo $response->getBody();
```
</RequestExample>

## Parameters

<ParamField path="client_id" type="string" required>
  Your merchant API client/primary key obtained from the merchant dashboard
</ParamField>

<ParamField path="secret_id" type="string" required>
  Your merchant API secret key obtained from the merchant dashboard
</ParamField>

## Response

### Success Response (200 OK)

<ResponseField name="message" type="object">
  <ResponseField name="code" type="number">200</ResponseField>
  <ResponseField name="success" type="array">
    <ResponseField name="0" type="string">SUCCESS</ResponseField>
  </ResponseField>
</ResponseField>

<ResponseField name="data" type="object">
  <ResponseField name="access_token" type="string">
    JWT-style access token for authenticating subsequent requests
  </ResponseField>
  <ResponseField name="expire_time" type="number">
    Token expiration time in seconds (600 seconds)
  </ResponseField>
</ResponseField>

<ResponseField name="type" type="string">success</ResponseField>

### Error Response (400 Bad Request)

<ResponseField name="message" type="object">
  <ResponseField name="code" type="number">400</ResponseField>
  <ResponseField name="error" type="array">
    <ResponseField name="0" type="string">Invalid secret ID</ResponseField>
  </ResponseField>
</ResponseField>

<ResponseField name="data" type="array">[]</ResponseField>
<ResponseField name="type" type="string">error</ResponseField>

## Error Codes

  The provided client_id or secret_id is invalid or incorrect
