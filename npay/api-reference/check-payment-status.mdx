# Check Payment Status

Check the current status of a payment transaction using the payment token returned from the initiate payment endpoint.

<Note>
  This endpoint allows you to verify payment completion and retrieve transaction details.
</Note>

## Request

<RequestExample>
```php
<?php
require_once('vendor/autoload.php');
$client = new \GuzzleHttp\Client();
$response = $client->request('GET', '{{base_url}}/payment/status/2zMRmT3KeYT2BWMAyGhqEfuw4tOYOfGXKeyKqehZ8mF1E35hMwE69gPpyo3e', [
    'headers' => [
        'Authorization' => 'Bearer {{access_token}}',
        'accept' => 'application/json',
    ],
]);
echo $response->getBody();
```
</RequestExample>

## Payment Statuses

The payment can have different statuses:

- **Pending** - Payment is being processed
- **Completed** - Payment was successful
- **Failed** - Payment failed
- **Cancelled** - Payment was cancelled by the user
- **Expired** - Payment token has expired

## Use Cases

### Verify Payment Completion

After a customer completes payment, use this endpoint to verify the transaction was successful:

```php
// Check if payment was completed
$status = checkPaymentStatus($paymentToken);
if ($status['data']['status'] === 'completed') {
    // Process successful payment
    processOrder($orderId);
}
```

### Retrieve Transaction Details

Get detailed information about a completed payment:

```php
$paymentDetails = getPaymentStatus($paymentToken);
$transactionId = $paymentDetails['data']['trx_id'];
$payerEmail = $paymentDetails['data']['payer']['email'];
```

### Handle Webhook Verification

Verify webhook notifications by checking payment status:

```php
// Verify webhook data
$webhookToken = $_POST['token'];
$paymentStatus = checkPaymentStatus($webhookToken);
// Process based on status
```
