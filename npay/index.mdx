## Welcome to NPAY Payment Gateway

NPAY is a comprehensive payment gateway solution that provides secure, reliable, and easy-to-integrate payment processing capabilities for your applications. Our API allows you to accept payments from customers worldwide with support for multiple currencies and payment methods.

## What You Can Do

- **Accept Payments**: Process payments securely through our payment gateway
- **Multiple Currencies**: Support for various international currencies
- **Real-time Status**: Check payment status in real-time
- **Secure Authentication**: OAuth2-based authentication for secure API access
- **Webhook Support**: Receive real-time notifications about payment events
