## Common Issues & Solutions

### Authentication Problems

**Issue**: Getting "Invalid credentials" error

**Solutions**:
1. Verify your Client ID and Secret Key
2. Check if you're using the correct environment
3. Ensure credentials haven't been regenerated
4. Test with a fresh access token

### Payment Creation Fails

**Issue**: Payment creation returning errors

**Solutions**:
1. Validate all required parameters
2. Check amount format (2 decimal places)
3. Verify currency code (uppercase, Alpha-3)
4. Ensure URLs are properly formatted
5. Confirm access token is valid

### Status Check Issues

**Issue**: Can't retrieve payment status

**Solutions**:
1. Verify payment token is correct
2. Check if token has expired
3. Ensure access token is valid
4. Confirm payment exists
